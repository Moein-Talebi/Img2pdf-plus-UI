# ✨ Img2pdf Plus UI

A modern, "premium" web application designed for lightning-fast and private image-to-PDF conversion. Built with a focus on aesthetics and user experience.

![Vite](https://img.shields.io/badge/vite-%23646CFF.svg?style=for-the-badge&logo=vite&logoColor=white)

## 🚀 Key Features

- **💎 Premium Design**: Sleek, dark-mode interface featuring glassmorphism elements and smooth transitions.
- **🔒 Privacy First**: 100% client-side processing. Your images never leave your browser or touch a server.
- **⚡ Ultra Fast**: Near-instant PDF generation using the `jsPDF` engine.
- **🛠️ Smart Filtering**: Built-in options to filter images based on minimum and maximum file size (MB) before conversion.
- **🖼️ Live Preview**: Interactive preview grid with the ability to review and remove images before final export.
- **🖱️ Drag & Drop**: Seamless upload experience with interactive visual feedback.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **State Management**: React Hooks
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **PDF Generation**: [jsPDF](https://github.com/parallax/jsPDF)
- **Styling**: Vanilla CSS with custom design tokens

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Moein-Talebi/Img2pdf-plus-UI.git
   cd Img2pdf-plus-UI
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Access the application**:
   Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🛡️ Privacy Guarantee

This tool processes everything locally. No images are uploaded, stored, or processed on any remote server. Your data stays on your machine.
